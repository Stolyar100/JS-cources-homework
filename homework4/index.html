Відмінності між стрілковими та звичайними функціями

  • Синтаксис

    Стрілкові функції записуються простіше:

                        // (param1, param2, paramN) => expression
                    // ES5
                    var add = function(x, y) {
                        return x + y;
                    };
                    // ES6
                    let add = (x, y) => { return x + y };

  • Прив'язка аргументів

    У стрілкових функцій немає аргументів. Однак вони мають доступ до аргументів об'єкта найближчої батьківської функції без стрілки:

                        let myFunc = {  
                        showArgs(){ 
                            console.log(arguments); 
                        } 
                    }; 
                    myFunc.showArgs(1, 2, 3, 4);
                
                        let myFunc = {  
                        showArgs : () => { 
                            console.log(...arguments); 
                        } 
                    }; 
                    myFunc.showArgs(1, 2, 3, 4);

  • використання ключового слова this

    У стрілкової функції немає власного this:

                        let me = { 
                        name: "Ashutosh Verma", 
                        thisInArrow:() => { 
                        console.log("My name is " + this.name); // no 'this' binding here 
                        }, 
                        thisInRegular(){ 
                        console.log("My name is " + this.name); // 'this' binding works here 
                        } 
                    };
                    me.thisInArrow(); 
                    me.thisInRegular();

  • використання ключового слова new

    Стрілкову функцію не вийде використати як конструктрок з ключовим словом new:

                        let add = (x, y) => console.log(x + y);
                    new add(2,3);

  • Не можна дублювати аргументи

    Аргументи стрілкової функції не можна дублювати, як і не можна дублювати аргументи звичайної в 'use strict':

                        (x, x) => {}
                    // SyntaxError: duplicate argument names not allowed in this context
